# LexiFlow AI Backend

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Environment Variables:
    The `.env` file is already configured with the database connection string.

## Running

-   **Development:**
    ```bash
    npm run dev
    ```

-   **Production Build:**
    ```bash
    npm run build
    npm start
    ```

## API Endpoints

-   `GET /`: Health check
-   `GET /db-test`: Test database connection
